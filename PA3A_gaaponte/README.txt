How to run program in command line:
	Download PA3A_gaaponte.zip and unzip it. Open the terminal and navigate to the folder where you unzipped the folder. 
Type the comand "make into the command line. This will compile the program. Now, navigate to the to the saem folder
inside the terminal again, but this time type the command "./ctest". This will run the program.

How to run program in eclipse:
	Download PA3A_gaaponte.zip and unzip it. Open eclipse snd click on the File within the header on the top
left. Then click on "Import..". You will be presented with a drop down menue. Click on "General" and then
"Existing projects into workspace". Once you do this, browse the directory and find the unziped file and
select it. Now you can build the project and click run. 

Description of program:
	mystring.c contains variations of C Library String functions that I wrote to show how they work. ctest.c
contains many tests for each of these functions to prove that they work just like the original functions.
